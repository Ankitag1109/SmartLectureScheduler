# Smart Lecture Scheduler

Smart Lecture Scheduler is a full-stack web application built with **React** (frontend), **Node.js + Express** (backend), and **MongoDB** (database).  
It allows admins to manage instructors, courses, and lectures, while instructors can view their assigned lectures.

---

## Features

### Admin

- Admin login with authentication
- Dashboard showing total instructors, courses, and lectures
- Add, view, and delete instructors
- Add, view, and delete courses
- Add lectures and assign them to courses
- View all lectures

### Instructor

- Instructor login with authentication
- View assigned lectures in a table format
